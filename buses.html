Buses

What are buses?

Buses are the wires/connections which join together different components of the CPU (and the wider computer) allowing them to communicate, sending data all around the computer.

Which buses are present in the CPU?

Address Bus

The address bus is used to tell other register which memory address in the RAM is currently being read from or written to.

Data Bus

The data bus is the main bus used to move data around the CPU, generally from the memory to MDR or back again.
