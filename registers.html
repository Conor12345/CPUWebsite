Registers

What is a register?

A register is a small amount of memory. The CPU has many of these small "chunks" of memory, each with a different purpose.

What types of register are in a CPU?

Name Abbreviation Function
Memory Data Register MDR This is where data from the RAM is stored to be used by the ALU in the CPU. Only data is stored here, any CPU instructions are stored in CIR.
Current Instruction Register CIR If the data from memory is deemed to an instruction, it is put into to CIR. This is how the other CPU components know what to do with the data.
Memory Address Register MAR This is the register which stores the memory address which the buses needed to move to the MDR or CIR. Each memory location has its own address which the address bus can select, allowing the data bus to retrieve the data. This process is also reversed to store data into memory.
Status Register SR The status register stores information about the CPU as a whole to ensure everything is co-ordinated and working correctly. If any of the components of the CPU have an error the status register is changed to alert the rest of the CPU components.
Program Counter PC The program counter is used to keep the whole CPU synchronized around the same memory address. The PC increments by one every time the CPU is finished with a memory location to instruct the process to being again on the next memory location.
